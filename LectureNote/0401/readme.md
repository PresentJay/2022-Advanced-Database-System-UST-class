# 부질의

## SQL Join Expressions

- 조인 수식은 그 자체가 질의일 수도 있고, 또는 부질의로도 사용될 수 있다.
- `CROSS JOIN` = `Cartesian Product`
- `INNER JOIN` ... ON
- 외부조인 (outerjoin)
    - 허상 튜플을 결과에 추가
        - 허상 튜플 (dangling tuple): 조인되지 못한 튜플
        - 조인되지 않은 경우, Attribute들을 NULL 값으로 처리
    - `LEFT` | `RIGHT`
